## Full Description (max 4000 characters)

**Transfer files instantly between devices without internet**

FileDrop makes sharing files between your devices simple, fast, and secure. No cloud storage, no data limits, no internet connection required – just direct peer-to-peer transfer over your local Wi-Fi network.

**Key Features:**

✓ **Direct Transfer** – Files go straight from sender to receiver, never touching any server
✓ **No Internet Required** – Works on your local Wi-Fi network
✓ **Fast & Secure** – WebRTC encryption ensures your files stay private
✓ **No File Size Limits** – Transfer files of any size
✓ **Easy to Use** – Scan QR code or enter a simple code to connect
✓ **Privacy First** – Zero data collection, no tracking, no ads

**How It Works:**

1. One device creates a session and displays a QR code
2. The other device scans the code or enters the session code
3. Select files and send – that's it!

**Perfect For:**

• Sharing photos and videos between your phone and tablet
• Transferring documents between devices
• Moving files without using cloud storage
• Quick file sharing when you don't have internet access

**Privacy & Security:**

FileDrop is built with privacy as the top priority. All transfers use encrypted WebRTC connections. No files, personal data, or device information is ever collected or stored on any server. Session codes expire after 10 minutes for added security.

**Open Source & Transparent:**

FileDrop is built on open web technologies and runs on the Internet Computer blockchain for decentralized, trustless operation.

**Technical Details:**

• Minimum Android version: 7.0 (API 24)
• Permissions: Camera (for QR scanning), Storage (for file selection)
• Network: Local Wi-Fi or internet connection for WebRTC signaling
• No account required
